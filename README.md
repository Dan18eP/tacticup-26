# TactiCup 26

Where strategy defines the game.

## Academic subtitle
Interactive World Cup Simulator 2026

## Overview
TactiCup 26 is a web-based football simulation project focused on tactical decision-making and real-time visual match events.

## Tech stack
- React + Vite
- Node.js + Express
- Server-Sent Events (SSE)
- JavaScript
- CSS

## Repository structure
- `frontend/`: user interface
- `backend/`: API and simulation engine
- `docs/`: project documentation
